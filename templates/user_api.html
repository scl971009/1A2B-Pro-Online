1A2B Pro Online User API List

Every api can use data.exception to get error message if any error occurs


Sign up

/sign_up/<account>/<password>/<name>

status_code = data.result, account = data.account, name = data.name

0 (success) / 1 (account not unique) / -1 (unknown error)


Get Password

/get_login/<account>

password = data.password

(if account not existed, data.password == 0)


Get Score / Win / Lose

/get_score/<account>

score = data.score, win(PVP) = data.win_p, lose(PVP) = data.lose_p, win(PVE) = data.win_e, lose(PVE) = data.win_e

(if account not existed, data.score == data.win_p == data.lose_p == data.win_e == data.lose_e == -1)


Note that the logic of add win and add lose is

Enter game, add lose to both players.

Game ended, add win to the winner.


Add Win (PVP)

/pvp/add_win/<account>

status_code = data.result

0 (success) / -1 (error)


Add Lose (PVP)

/pvp/add_lose/<account>

status_code = data.result

0 (success) / -1 (error)


Add Win (PVE)

/pve/add_win/<account>

status_code = data.result

0 (success) / -1 (error)


Add Lose (PVE)

/pve/add_lose/<account>

status_code = data.result

0 (success) / -1 (error)


Select rival

/online/<account>/<rival1's account>/<rival2's account>/...

rival = data.rival

(if account or none of the rivals' accounts not existed, data.rival == 0)
