1A2B Pro Online User API List

Sign up

/sign_up/<account>/<password>/<name>

status_code = data.result, account = data.account, name = data.name

0 (success) / 1 (account not unique) / 2 (field(s) not all filled) / -1 (unknown error)


Get Password

/get_login/<account>

password = data.password

(if account not existed, data.password == 0)


Get Score / Win / Lose

/get_score/<account>

score = data.score, win = data.win, lose = data.lose

(if account not existed, data.score == data.win == data.lose == -1)


Note that the logic of add win and add lose is

Enter game, add lose to both players.

Game ended, add win to the winner.


Add Win

/add_win/<account>

status_code = data.result

0 (success) / -1 (error)


Add Lose

/add_lose/<account>

status_code = data.result

0 (success) / -1 (error)
